# Alif Laila Introductory Astronomy

This site hosts materials for the **Alif Laila Introductory Astronomy** program,
including observatory design documentation, instructional lessons, and practice
materials for students and outreach participants.

<div class="callout">
This project is designed for open-access education and remote astronomical
observation using a roll-off roof observatory.
</div>
